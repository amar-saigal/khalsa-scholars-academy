# Khalsa Scholars Academy — College Prep Program (CCP)

A community-driven academic excellence initiative designed to help students build strong foundations in Math, English, and Science while preparing progressively for PSAT, SAT, ACT, and National Merit pathways.

## Purpose

Khalsa Scholars Academy is being designed as an **academic acceleration system**, not a generic tutoring program.

The goal is to help students:

- stay ahead of school curriculum
- build mastery and confidence
- develop disciplined study habits
- receive community-based mentorship and accountability
- grow into future peer leaders and mentors

The program is rooted in Sikh values including:

- Seva
- Charhdi Kala
- Sarbat da Bhala
- discipline
- humility
- peer support

## Current Stage

This repository currently holds the **design system** for the program:

- reusable prompts for LLM-assisted design
- program configuration
- operating assumptions
- draft program documents
- pilot planning materials

The MVP / pilot is expected to begin with a narrow, practical scope and expand over time.
